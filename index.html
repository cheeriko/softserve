Getting Started

I made this for me, mostly, and for other people who aren't super code savvy but need to throw together a site for whatever reason. A nice boilerplate to work on but also a good enough framework to just make a whole (lite) site with.

No need for compilers, it's just straight CSS (although it might look jank in some browsers. Fuck IE8.)

Uses CSS variables so you can quickly style and go.

Inspired by and utilized techniques from Milligram.CSS, Chota.CSS, and the Bootstrap 4 Framework.
